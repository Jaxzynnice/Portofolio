# Jaxzynnice-UI | Portfolio Website

## 🚀 About
Modern portfolio website for Januar Adi Surya Wijaya (Jaxzynnice) - WhatsApp Bot & Web Scraping Specialist. Features glassmorphism design, smooth animations, and responsive layout.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Glassmorphism, Flexbox, Grid)
- Vanilla JavaScript
- Font Awesome Icons

## ✨ Features
- Glassmorphism UI effects
- Responsive design (Mobile, Tablet, Desktop)
- Skeleton loading animation
- Tech stack carousel
- Typing animation
- Interactive chat simulation
- Smooth scroll animations
- Number counter animation
- Custom cursor effects

## 📁 Project Structure
```

jaxzynnice-ui/
├── index.html          # Main HTML structure
├── css/
│   ├── main.css        # Core styles
│   ├── glassmorphism.css # Glass effects
│   └── responsive.css  # Responsive design
├── js/
│   ├── main.js         # Core functionality
│   ├── carousel.js     # Tech stack carousel
│   ├── skeleton.js     # Loading skeleton
│   └── animations.js   # Scroll & counter animations
└── README.md

```
